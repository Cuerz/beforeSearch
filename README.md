# beforeSearch

## 简介

插件为manifest v3版本开发，使用了vue3+element-plus

最终预期实现功能包括多种编码转换，资产测绘，指纹识别，前端敏感信息发现，信息搜集，等等，后续可能会增加更多功能。

## 使用方式

下载release新版本，将解压后的dist文件夹拖入到谷歌浏览器插件页面即可
